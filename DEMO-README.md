# 璟程 APP - 招商演示版

## 🎯 项目定位

**AI 智能酒店预订平台** - 新一代酒店预订 + 会员分红一站式解决方案

---

## ✨ 核心功能

### 1. 智能酒店预订
- 🏨 酒店/民宿/公寓/青旅全覆盖
- 🔍 智能搜索 + 多维度筛选
- 📅 实时房态 + 在线预订
- 💳 多种支付方式

### 2. AI 智能助手
- 🤖 24 小时在线服务
- 💬 智能对话推荐
- 🎯 个性化酒店推荐
- ❓ 常见问题自动解答

### 3. 会员系统
- 👑 多级会员体系
- 💎 积分累积 + 消费
- 🔗 邀请返利机制
- 💰 分享赚钱功能

### 4. 订单管理
- 📋 订单列表 + 详情
- ✅ 实时状态更新
- ❌ 在线取消/修改
- 📄 电子发票支持

### 5. 个人中心
- 👤 用户信息管理
- 📊 消费记录统计
- 🎁 优惠券管理
- ⚙️ 设置功能

## 📱 技术架构

### 前端
- **框架**: Vue 3 (Composition API)
- **UI**: 自定义组件库
- **移动端**: Capacitor 7
- **PWA**: Service Worker

### 后端 (可选配)
- **Node.js** + Express
- **数据库**: MySQL / MongoDB
- **缓存**: Redis
- **消息**: RabbitMQ

### 部署
- **Web**: Vercel / Netlify
- **Android**: APK / 应用商店
- **iOS**: IPA / TestFlight / App Store
- **小程序**: 微信小程序 (可选)
