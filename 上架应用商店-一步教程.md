# 璟程 APP - 上架应用商店一步教程

**最简单的方式，跟着做就行！**

---

## 🎯 您需要做的事（按顺序）

### 第 1 步：获取 APK 安装包（15 分钟）

**使用 Ionic Appflow 在线构建**

1. **访问 Ionic**
   ```
   https://ionic.io/appflow
   ```

2. **注册账号**
   - 点击「Get Started」
   - 用邮箱注册（免费）

3. **创建项目**
   - 登录后点击「Create Project」
   - 选择「Capacitor」
   - 名称：`璟程 APP`

4. **上传代码**
   - 方式 A（推荐）：用 Git
     ```bash
     cd /home/admin/openclaw/workspace/jingcheng-app
     git init
     git add .
     git commit -m "璟程 APP"
     # 然后 push 到 GitHub，在 Ionic 连接 GitHub
     ```
   
   - 方式 B：用 Ionic CLI
     ```bash
     npm install -g @ionic/cli
     ionic login
     cd /home/admin/openclaw/workspace/jingcheng-app
     ionic upload
     ```

5. **配置签名**
   - 进入项目 Settings
   - 找到「Android Signing」
   - 上传签名文件：`android/jingcheng.jks`
   - 填写：
     - Alias: `jingcheng`
     - Password: `jingcheng2026`

6. **开始构建**
   - 点击「Builds」→「New Build」
   - Platform: Android
   - Build Type: Release
   - 点击「Run Build」
   - 等待 5-10 分钟

7. **下载 APK**
   - 构建完成后点击「Download」
   - 保存到电脑

**完成！** 您现在有了 APK 文件。

---

### 第 2 步：准备上架材料（30 分钟）

**需要的文件**:

1. **APK 文件**
   - 刚从 Ionic 下载的
   - 约 30-50MB

2. **应用图标**
   - 位置：`www/icon-512.png`
   - 尺寸：512x512
   - 格式：PNG

3. **应用截图**（5-8 张）
   - 用手机横屏截图
   - 或从设计稿导出
   - 尺寸：1080x1920 或更大
   - 内容：
     - 首页
     - 酒店列表
     - 酒店详情
     - 预订页面
     - 订单管理
     - AI 助手
     - 个人中心

4. **应用描述**（复制这段）
```

• 在线预订 - 简单快捷的预订流程
• 订单管理 - 实时查看订单状态
• 会员系统 - 积分累积、等级特权
• 邀请返利 - 分享赚钱、收益提现

【设计特色】
• 深空蓝 + 鎏金色奢华设计
• 流畅的交互动画
• 沉浸式使用体验

【技术优势】
• Vue 3 最新技术
• 秒级加载速度
• 智能推荐算法
• 数据安全保障

```

5. **隐私政策**（复制这段）
```

1. 采用 HTTPS 加密传输

1. 访问、更正个人信息
2. 删除个人信息
3. 注销账号
```

6. **资质文件**
   - 营业执照（企业账号需要）
   - ICP 许可证（如有）
   - 软件著作权（如有，建议办理）

---

### 第 3 步：注册开发者账号（10 分钟）

**小米应用商店**（推荐先上这个）

1. **访问**
   ```
   https://dev.mi.com
   ```

2. **注册账号**
   - 点击「注册」
   - 用手机号注册

3. **实名认证**
   - 选择「企业开发者」
   - 上传营业执照
   - 填写企业信息
   - 等待审核（通常当天通过）

**华为应用市场**（可选）

1. **访问**
   ```
   https://developer.huawei.com
   ```

2. **注册并认证**
   - 类似小米流程

---

### 第 4 步：提交上架（20 分钟）

**以小米为例**:

1. **登录小米开发者后台**
   ```
   https://dev.mi.com/console
   ```

2. **创建应用**
   - 应用开发 → 应用发布
   - 点击「创建新应用」

3. **填写基本信息**
   - 应用名称：`璟程`
   - 包名：`com.jingcheng.app`
   - 分类：`旅游出行` → `酒店预订`
   - 一句话简介：`AI 智能酒店预订平台`

4. **上传 APK**
   - 点击「上传 APK」
   - 选择从 Ionic 下载的 APK
   - 等待上传完成

5. **上传图标**
   - 点击「上传图标」
   - 选择 `www/icon-512.png`

6. **上传截图**
   - 点击「添加截图」
   - 上传 5-8 张截图

7. **填写描述**
   - 复制上面的应用描述
   - 粘贴到描述框

8. **上传资质**
   - 营业执照
   - ICP 许可证（如有）

9. **填写隐私政策**
   - 复制上面的隐私政策
   - 或上传隐私政策页面链接

10. **提交审核**
    - 检查所有信息
    - 点击「提交审核」

**完成！** 等待审核通过（1-3 天）

---

### 第 5 步：等待审核（1-3 天）

**审核期间**:
- 保持手机畅通（可能接到审核电话）
- 关注审核状态
- 如有驳回，根据反馈修改

**审核通过后**:
- 应用自动上线
- 用户可在小米应用商店搜索下载
- 收到上线通知

---

### 第 6 步：上架其他商店（可选）

重复第 3-5 步，上架到：
- 华为应用市场
- OPPO 软件商店
- vivo 应用商店
- 应用宝（腾讯）

## 💡 重要提示

### 关于资质
- **个人开发者**: 只能上架少量商店
- **企业开发者**: 所有商店都可以
- **ICP 许可证**: 部分商店必需
- **软件著作权**: 加分项，非必需

### 关于审核
- **首次审核**: 较严格，1-3 天
- **更新审核**: 较快，几小时
- **被拒**: 根据反馈修改后重新提交

### 关于费用
- **开发者账号**: 免费（除苹果$99/年）
- **上架费用**: 免费
- **推广费用**: 可选（应用商店广告）
