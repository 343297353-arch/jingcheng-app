## 🔧 您需要手动完成的步骤

### 第 1 步：推送代码到 GitHub（5 分钟）

1. **注册/登录 GitHub**
   - 访问：https://github.com
   - 如果没有账号，先注册

2. **创建新仓库**
   - 点击右上角「+」→「New repository」
   - 名称：`jingcheng-app`
   - 设为 Public 或 Private 都可以
   - 点击「Create repository」

3. **推送代码**
   ```bash
   cd /home/admin/openclaw/workspace/jingcheng-app
   git remote add origin https://github.com/你的用户名/jingcheng-app.git
   git push -u origin master
   ```

### 第 2 步：Ionic Appflow 构建 APK（15 分钟）

1. **注册 Ionic**
   - 访问：https://ionic.io/appflow
   - 点击「Get Started」
   - 用邮箱注册（免费）

2. **创建项目**
   - 登录后点击「Create Project」
   - 选择「Import from Git」
   - 连接 GitHub，选择 `jingcheng-app` 仓库

3. **配置签名**
   - 进入项目 Settings → Android Signing
   - 上传签名文件：`/home/admin/openclaw/workspace/jingcheng-app/android/jingcheng.jks`
   - 填写：
     - Alias: `jingcheng`
     - Password: `jingcheng2026`

4. **开始构建**
   - 点击「Builds」→「New Build」
   - Platform: Android
   - Build Type: Release
   - 点击「Run Build」
   - 等待 5-10 分钟

5. **下载 APK**
   - 构建完成后点击「Download」
   - 保存到电脑

1. **访问**
   ```
   https://dev.mi.com
   ```

2. **注册账号**

   - 用手机号注册

3. **实名认证**

A: 部分商店需要，小米暂时不需要。建议后续办理。
