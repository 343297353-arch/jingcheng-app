# 璟程 APP - 一键打包指南

## 🎯 快速打包（无需配置环境）

由于服务器没有 Android SDK，我们使用**在线构建服务**，这是最简单的方式。

---

## ✅ 方案一：Ionic Appflow（推荐）

### 步骤：

1. **访问 Ionic Appflow**
   - 网址：https://ionic.io/appflow
   - 注册免费账号

2. **创建项目**
   - 点击"Create Project"
   - 选择"Capacitor"
   - 项目名称：璟程 APP

3. **上传代码**
   - 方式 A: 连接 GitHub
   - 方式 B: 直接上传 zip 文件
   - 上传：`jingcheng-deploy-package.zip`

4. **配置签名**
   - 进入 Settings → Android Signing
   - 上传：`android/jingcheng.jks`
   - 密码：`jingcheng2026`
   - 别名：`jingcheng`

5. **开始构建**
   - 点击"Build"
   - 选择"Release"
   - 等待 5-10 分钟
   - 下载 APK 文件

### 费用：
- 免费版：每月 5 次构建
- 付费版：$19/月，无限构建

## ✅ 方案二：Codemagic

### 步骤：

1. **访问 Codemagic**
   - 网址：https://codemagic.io
   - 用 GitHub 登录

2. **创建项目**
   - Add repository
   - 选择 Capacitor 项目

3. **配置构建**
   ```yaml
   workflows:
     android:
       name: 璟程 Android
       instance_type: mac_mini_m1
       max_build_duration: 30
       environment:
         groups:
           - jingcheng_signing
       scripts:
         - npm install
         - cd android
         - ./gradlew assembleRelease
       artifacts:
         - android/app/build/outputs/apk/release/*.apk
   ```

4. **添加签名**
   - Settings → Code Signing
   - 上传 keystore 文件
   - 设置环境变量

5. **开始构建**
   - 点击"Start new build"
   - 等待完成
   - 下载 APK

### 费用：
- 免费版：每月 500 分钟
- 付费版：$29/月

## ✅ 方案三：本地打包（如果你有 Mac/PC）

### 下载完整包：

```bash
# 下载打包文件
cd /home/admin/openclaw/workspace/jingcheng-app
ls -lh jingcheng-deploy-package.zip
```

### Android 打包：

1. **安装 Android Studio**
   - 下载：https://developer.android.com/studio
   - 安装并配置 SDK

2. **打开项目**
   ```bash
   cd jingcheng-app/android
   # 用 Android Studio 打开 android 目录
   ```

3. **配置签名**
   - File → Project Structure
   - Build Variants → signingConfigs
   - 选择 `jingcheng.jks`

4. **构建 APK**
   - Build → Generate Signed Bundle / APK
   - 选择 APK
   - 选择 release
   - 点击 Finish

5. **获取 APK**
   - 位置：`android/app/build/outputs/apk/release/app-release.apk`

### iOS 打包（需要 Mac）：

1. **安装 Xcode**
   - App Store 下载 Xcode
   - 安装命令行工具

2. **打开项目**
   ```bash
   open ios/App/App.xcworkspace
   ```

3. **配置签名**
   - 选择开发团队
   - Bundle ID: com.jingcheng.app

4. **Archive**
   - Product → Archive
   - 等待完成

5. **导出 IPA**
   - Distribute App
   - 选择 Ad Hoc 或 App Store
   - 导出 IPA 文件

## 📱 应用市场上架指南

### 国内 Android 应用商店

#### 1. 小米应用商店
- **网址**: https://dev.mi.com/console/appservice/publish.html
- **费用**: 免费
- **审核时间**: 1-3 天
- **需要资质**: 
  - 营业执照
  - ICP 许可证
  - 软件著作权（可选）

#### 2. 华为应用市场
- **网址**: https://developer.huawei.com/consumer/cn/service/jsp/agreement/publish.jsp
- **费用**: 免费
- **审核时间**: 1-3 天
- **需要资质**:
  - 营业执照
  - ICP 许可证

#### 3. OPPO 软件商店
- **网址**: https://open.oppomobile.com/
- **费用**: 免费
- **审核时间**: 1-2 天

#### 4. vivo 应用商店
- **网址**: https://dev.vivo.com.cn/
- **费用**: 免费
- **审核时间**: 1-2 天

#### 5. 应用宝（腾讯）
- **网址**: https://open.qq.com/
- **费用**: 免费
- **审核时间**: 2-5 天

### 上架流程：

1. **注册开发者账号**
   - 准备营业执照
   - 完成实名认证

2. **准备材料**
   - APK 文件
   - 应用截图（5-8 张）
   - 应用描述（300 字以内）
   - 图标（512x512）
   - 隐私政策链接

3. **提交审核**
   - 填写应用信息
   - 上传 APK
   - 提交审核

4. **等待上线**
   - 审核通过后自动上线
   - 被拒根据反馈修改

## 🍎 iOS App Store 上架

### 准备工作：

1. **Apple Developer 账号**
   - 费用：$99/年
   - 网址：https://developer.apple.com
   - 需要公司资质

2. **App Store Connect**
   - 创建 App ID
   - 创建证书
   - 创建 Provisioning Profile

3. **准备材料**
   - IPA 文件
   - 应用截图（6.5 寸和 5.5 寸）
   - 应用描述
   - 关键词
   - 隐私政策

### 上架流程：

1. **创建 App**
   - 登录 App Store Connect
   - 创建新 App
   - 填写基本信息

2. **上传 IPA**
   - 用 Xcode Archive
   - 或 Transporter 上传

3. **填写信息**
   - 应用截图
   - 描述
   - 关键词
   - 分类

4. **提交审核**
   - 选择发布方式
   - 提交审核
   - 等待 24-48 小时

5. **发布**
   - 审核通过后发布
   - 或手动发布
