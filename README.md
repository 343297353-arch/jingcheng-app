# 璟程 - AI 智能酒店预订平台 v2.0

> 高端酒店预订 APP | AI 智能助手 | 会员分红系统

## 📱 项目简介

**璟程** 是一款高端酒店预订应用，整合酒店、民宿、公寓、青旅等多种住宿类型，配备 AI 智能助手，提供个性化推荐和 24 小时在线客服。

### 核心特色

- 🏨 **全类型住宿** - 酒店/民宿/公寓/青旅全覆盖
- 🤖 **AI 智能助手** - 24 小时在线，智能推荐酒店
- 💎 **会员体系** - 四级会员 + 积分 + 分红系统
- 🎨 **高端 UI** - 深空蓝 + 鎏金色，商务奢华风格
- 📱 **PWA 支持** - 可安装到桌面，离线可用

## 📦 封装成 APP

### 方案 A：Capacitor（推荐）

```bash
# 安装 Capacitor
npm install -g @capacitor/cli @capacitor/core

# 初始化
npx cap init

# 添加平台
npx cap add android
npx cap add ios

# 构建
npx cap sync

# 打开 Android Studio / Xcode
npx cap open android
npx cap open ios
```

### 方案 B：PWA 直接安装

1. 部署到 HTTPS 服务器
2. 用户访问网站
3. 浏览器提示"添加到主屏幕"
4. 即可像 APP 一样使用

## 📱 上架应用商店

### 需要准备的材料

1. **应用截图** - 多尺寸（6.5 寸、5.5 寸、平板）
2. **应用描述** - 简介、特性、更新说明
3. **隐私政策** - 用户数据使用说明
4. **用户协议** - 服务条款
5. **营业执照** - 支付接口需要
6. **软件著作权** - 建议申请

### 上架流程

1. 用 Capacitor 封装成 IPA/APK
2. 注册开发者账号（苹果$99/年，安卓大部分免费）
3. 准备上架材料
4. 提交审核
5. 等待审核通过（苹果 1-3 天，安卓 1-7 天）
